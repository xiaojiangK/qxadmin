MessageBox 弹框

模拟系统的消息提示框而实现的一套模态对话框组件，用于消息提示、确认消息和提交内容。

消息提示

当用户进行操作时会被触发，该对话框中断用户操作，直到用户确认知晓后才可关闭。

点击打开 Message Box

确认消息

提示用户确认其已经触发的动作，并询问是否进行此操作时会用到此对话框。

点击打开 confirm Box

自定义内容

可自定义配置不同内容。

点击打开 Custom Box

提交内容

当用户进行操作时会被触发，中断用户操作，提示用户进行输入的对话框。

点击打开 Submit Box

Message-box Attributes

参数 说明 类型 可选值 默认值
title MessageBox 标题 string — 提示
message MessageBox 消息正文内容 string / VNode — 这是一段内容
type 消息类型，用于显示图标 string success / info / warning / error —
showInput 是否显示输入框 boolean — false
placeholder 输入框的占位符 string — —
type 输入框的类型 string — text
value 输入框的初始文本 string — —
maxLength 输入框的最大值 number — —
minLength 输入框的最小值 number — —
showCancelBtn 是否显示取消按钮 boolean — false
showConfirmBtn 是否显示确定按钮 boolean — true
cancelBtnText 取消按钮的文本内容 string — 取消
confirmBtnText 确定按钮的文本内容 string — 确定

Message-box Events

事件名称 说明 回调参数
confirm 点击确认时触发的事件 —
cancel 点击取消时触发的事件 —
close 点击关闭时触发的事件 —
