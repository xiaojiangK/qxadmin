Select 选择器

当选项过多时，使用下拉菜单展示并选择内容。

基础用法

适用广泛的基础单选

带默认值

Select 下拉框默认选中状态。

禁用状态

Select 下拉框不可用状态。

图标选择器

Select 下拉选择对应图标

可搜索

可以利用搜索功能快速查找选项

基础多选

适用性较广的基础多选，用 Tag 展示已选项

Select Attributes

参数 说明 类型 可选值 默认值
