Message 消息提示

常用于主动操作后的反馈提示。与 Notification 的区别是后者更多用于系统级通知的被动提醒。

基础用法

从顶部出现，3 秒后自动消失。

打开消息提示

不同状态

用来显示「成功、警告、消息、错误」类的操作反馈。

成功 警告 消息 错误

可关闭

可以添加关闭按钮。

成功 警告 消息 错误

Message Attributes

参数 说明 类型 可选值 默认值
message 消息文字 string — —
type 主题 string success/warning/info/error info
duration 显示时间, 毫秒 number — 2000
showClose 是否显示关闭按钮 boolean — false
